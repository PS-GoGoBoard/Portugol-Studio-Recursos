Tipo Inteiro

Os números inteiros são de 32 bits com completo de dois constituidos dos números naturais positivos {1,2,3, ...}, negativos {-1, -2, -3, ...} e zero {0}.

O tipo de dado inteiro pode ser representado nas notações decimal, hexadecimal e binária.

Exemplo:

programa
{
    funcao inicio()
    {
        //imprime o número inteiro 32
        //escrito em notação decimal
        escreva(32)
        escreva("\n")

        //imprime o numero inteiro 32 
        //escrito em notação hexadecimal
        escreva(0x20)
        escreva("\n")

        //imprime o número inteiro 32
        //escrito em notação binária
        escreva(0b00100000)
        escreva("\n")
    }
}
