Tipo Lógico

É um tipo de dado, usado em operações lógicas, que possui somente dois valores, que são consideradas pelo Portugol como verdadeiro e falso

Exemplo:

programa
{
    funcao inicio()
    {
        //Imprime o valor verdadeiro do tipo lógico
        escreva(verdadeiro)
        //Imprime o valor falso do tipo lógico
        escreva(falso)		
        //Imprime o resultado da expressao lógica 
        escreva(verdadeiro ou falso) 
    }
}
