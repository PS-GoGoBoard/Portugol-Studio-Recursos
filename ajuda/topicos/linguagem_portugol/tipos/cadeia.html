Tipo Cadeia

Cadeia é uma sequência ordenada de caracteres (símbolos) escolhidos a partir de um conjunto pré-determinado.

O tipo cadeia deve estar acompanhado de aspas duplas.

Quando \n aprece em uma cadeia e gerada uma quebra de linha no texto

Exemplo:

programa
{
    funcao inicio()
    {
        //Imprime no console a frase Olá Mundo!
        escreva("Olá Mundo!")
        //Realiza uma quebra de linha
        escreva("\n")
        //Imprime texto com quebra de linha
        escreva("Texto exemplo \nde duas linhas.") 
    }
}
