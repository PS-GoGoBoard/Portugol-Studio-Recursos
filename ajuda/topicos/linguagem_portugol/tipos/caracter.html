Tipo Caracter

Caracter é uma unidade de informação que corresponde a um simbolo, exemplos de caracteres são letras, números e pontuações.

O tipo caracter deve estar acompanhado de aspas simples.

Exemplo:

programa
{
    funcao inicio()
    {
        //Imprime o caracter a
        escreva('a')
        //Imprime o caracter 1
        escreva('1')
        //Imprime o caracter ^ 
        escreva('^') 
    }
}
