Declaração de Matriz

Matriz é uma estrutura de dados simples similar ao vetor, porém possui uma dimensão a mais, o que permite organizar os valores em linha e coluna.

Para acessar um valor individual é necessário dois índices um para linha e outro para coluna.Durante a inicializadão deve-se respeitar a quantidade de linhas e colunas informado.

Sua declaração é bastante similar aos vetores, onde deve-se informar o tipo, seu nome, e a quantidade de linhas e colunas.

programa
{
    funcao inicio()
    {
        //Declaração de uma matriz de inteiros
        // de duas linhas e duas colunas já inicializado.
        inteiro matriz[2][2] = {{15,22},{10,11}}

        //Atribui -1 na primeira linha e segunda
        // coluna da matriz.
        matriz[0][1] = -1

        //Imprime o valor 15 correspontente 
        // a primeira linha e primeira coluna da matriz.
        escreva(matriz[0][0])
        escreva("\n")

        //Imprime o valor 11 correspontente 
        // a última linha e última coluna da matriz.
        escreva(matriz[1][1])

        //Declaração de uma matriz de reais de 
        // duas linhas e quatro colunas.
        real outra_matriz[2][4]

        //Declaração de uma matriz de caracteres onde o tamanho
        // de linhas e colunas são definidos pela inicialização
