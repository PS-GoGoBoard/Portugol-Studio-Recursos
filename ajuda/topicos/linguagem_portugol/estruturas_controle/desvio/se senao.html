Se-senao

Agora vamos imaginar que se a condição for falsa um outro conjunto de comandos deve ser executado. Quando iremos encontrar esta situação?

Imagine um programa onde um aluno com média final igual ou maior a 6 é aprovado. Se quisermos construir um algoritmo onde após calculada a média, seja mostrada na tela uma mensagem indicando se o aluno foi aprovado ou reprovado. Como fazer isto? Utilizando o comando se... senão! Olhe a sua sintaxe:


se (condicao) {
[conjunto de comandos]
}
senao{
[conjunto de comandos]
}


Veja que a se condição for verdadeira, os comandos localizados até o senão serão executados e o programa continuará após as chaves }. Já se a condição for falsa, os comandos entre senão e }{ é que serão executados e o programa prossegue sua execução.

A figura ilustra um algoritmo que verifica se o valor da hora digitada pelo usuário e armazenada na variável “hora”, corresponde a um horário do dia ou da noite. Para este exemplo, considerou-se que das 6 da manhã até as 18 horas é dia, caso contrário, já é noite. Ele faz isso usando um desvio condicional com senão. Note que o caminho seguindo pelo programa no caso do teste ser verdadeiro exibe uma mensagem, e no caso falso exibe outra mensagem , ou seja, só uma mensagem será exibida.




O exemplo a seguir ilustra em portugol o mesmo algoritmo do fluxograma acima.


Exemplo:

programa

        funcao inicio()
