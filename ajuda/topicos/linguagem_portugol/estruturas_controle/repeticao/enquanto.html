Laço Enquanto (Pré-Testado)

Nesta estrutura de repetição o programador deve definir uma condição que será verificada a cada iteração do loop B, caso essa condição seja verdadeira então é executado o bloco de código D e então a variável de controle deve ser alterada para evitar que o laço de repetição seja executado infinitamente.

Exemplo:

programa
{
    funcao inicio()
    {
        //Escreve no console o valor da variável passo durante as execuções
        inteiro passo = 0
        enquanto(passo < 5){
        	escreva(passo)
        	passo = passo + 1
        }

		passo = 0
        //Escreve no console o valor de passo multiplicado por 5
        enquanto (passo < 10){
        	escreva(passo*5)
        	passo = passo + 1
        }

        //Escreve no console "passo" enquanto teste for igual a verdadeiro
        passo = 0
        logico teste = verdadeiro
        enquanto(teste == verdadeiro){
        	escreva("passo")
        	passo = passo + 1
        	se (passo > 5){
        		teste = falso
        	}
        }	
    }
}
