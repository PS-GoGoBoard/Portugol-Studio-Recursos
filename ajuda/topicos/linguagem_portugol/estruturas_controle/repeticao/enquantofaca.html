Laço Faça-Enquanto (Pós-Testado)

Nesta estrutura de repetição o programador executa um bloco de instruções D e posteriormente faz uma verificação em B caso a condição de teste seja verdadeira executa o bloco de instrução até que a condição em B seja falsa.

Exemplo:

programa
{
    funcao inicio()
    {
        //Escreve no console o valor da variável passo durante as execuções
        inteiro passo = 0
        faca{
        	escreva(passo)
        	passo = passo + 1
        }enquanto(passo < 5)

		passo = 0
        //Escreve no console o valor de passo multiplicado por 5
        faca{
        	escreva(passo)
        	passo = passo + 1
        }enquanto(passo < 10)
        
        //Escreve no console "passo" enquanto teste for igual a verdadeiro
        passo = 0
        logico teste = verdadeiro
        faca{
        	escreva("passo")
        	passo = passo + 1
        	se (passo > 5){
        		teste = falso
        	}
        }enquanto(passo < 5)	
    }
}
