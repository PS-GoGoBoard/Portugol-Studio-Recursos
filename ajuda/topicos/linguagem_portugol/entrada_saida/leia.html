Leia

O Comando leia é utilizado quando se deseja obter informações do teclado do computador, ou seja, é um comando de entrada de dados. Esse comando aguarda um valor a ser digitado e o atribui diretamente na variável.

Para utilizar o comando leia, você deverá escrever este comando e entre parênteses colocar a(s) variavel (eis) que você quer que recebam os valores a serem digitados.

Exemplo:

programa
{
    funcao inicio()
    {
        inteiro numero

        //pede ao usuário que entre com um número
        escreva("Digite um valor inteiro: ")
        
        //lê o valor digitado pelo usuário
        leia(numero)

        //exibe a frase "O número digitado foi: " seguido do número digitado pelo usuário.
        escreva("O número digitado foi: ",numero)
    }
}
