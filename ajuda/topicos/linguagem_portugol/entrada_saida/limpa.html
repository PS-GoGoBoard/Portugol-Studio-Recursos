Limpa

Comando responsável por limpar o console. Não requer nenhum paramento e não tem nenhuma saída.

Exemplo:

programa
{
    funcao inicio()
    {
        //imprime a frase "Olá Mundo!"
        escreva("Olá Mundo!")
        
        //Limpa tudo que estava escrito no console 
        limpa()
    }
}
