Atribuições

Consiste em utilizar um valor obtido através da entrada de dados seja informado pelo usuário ou por através de uma constante, de modo que este valor seja armazenado por uma variável declarada ou um espaço de memória

Exemplo:

programa
{
    funcao inicio()
    {
        //Atribuição de valores constantes a uma variável
        inteiro a
        a = 2
        
        //Atribuição através de entrada de dados, informado pelo usuário
        inteiro b
        leia(b)
        
        //Atribuição através de uma variável já informada pelo usuário
        inteiro c
        c = b
    }
}
