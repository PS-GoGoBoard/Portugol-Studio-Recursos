Operações Aritméticas

As operações aritméticas tradicionais são a adição, a subtração, a multiplicação, a divisão e o módulo

A tabela a seguir demonstra o operador de cada operação aritmética disponível no Portugol:

Operador Descrição
+ Operação de adição
- Operação de subtração
* Operação de multiplicação
/ Operação de divisão
% operação de módulo
